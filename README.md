## 开发

### 拉取代码

```
git clone git@github.com:aa978563552/FEE-doc.git
```

### 启动

```
yarn docs:dev
```

### 部署

- 代码合并到主干 master 分支

- 本地执行脚本即可

```
sh deploy.sh
```
